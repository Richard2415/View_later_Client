# View-Later-Client

A MERN stack app to keep track of online stuff, which you may want to view later.

#### Env variable:

Create a .env file in server directory and add the following:

```
MONGODB_URI = "Your Mongo URI"
PORT = 3005
SECRET = "Your JWT secret"

```

#### Client:

```
cd client
npm install
npm start
```

#### Server:

```
cd server
npm install
npm run dev
```
